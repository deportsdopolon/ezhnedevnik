# Ежедневник — личный планировщик для iPhone

Веб-приложение (PWA): открываете в Safari и добавляете на экран «Домой» — работает как обычное приложение, без Mac и App Store.

**Ссылка:** https://deportsdopolon.github.io/ezhnedevnik/

## Установка на iPhone за 30 секунд

1. Откройте ссылку в **Safari** (не в Telegram/Chrome — именно Safari)
2. Нажмите кнопку **Поделиться** (квадрат со стрелкой вверх)
3. Выберите **«На экран "Домой"»**
4. Нажмите **Добавить**

Готово — иконка «Ежедневник» появится на рабочем столе.

## Возможности

- Недельный вид как на бумажном планере (RU/EN)
- Почасовые слоты 8:00–19:00
- Восход и закат, счётчик дня года
- Мини-календарь месяца
- Контакт к каждой записи (имя + телефон)
- Всё хранится локально на телефоне

## Как пользоваться

1. Нажмите на час (например `14`) — откроется запись
2. Введите заметку и контакт (имя/телефон)
3. Стрелки ← → — другая неделя, **Сегодня** — текущая неделя
