# Ежедневник — личный планировщик для iPhone

Недельный ежедневник в стиле бумажного планера: часы с 8 до 19, восход/закат, счётчик дня года, мини-календарь и привязка контактов из iPhone к каждой записи.

## Возможности

- Недельный вид как на бумажном планере (RU/EN)
- Почасовые слоты 8:00–19:00
- Восход и закат (по умолчанию — Москва, можно доработать под геолокацию)
- Счётчик «день года / дней до конца года»
- Мини-календарь месяца внизу
- Контакты iPhone: выбор человека к любой записи
- Данные хранятся локально на устройстве

## Что нужно

- Mac с **Xcode 15+**
- iPhone с **iOS 17+**
- Бесплатный Apple ID (для установки на свой телефон)
- Кабель USB / Wi‑Fi для подключения iPhone к Mac

> GitHub хранит **исходный код**. Сборка и установка на iPhone делаются через Xcode на Mac — это стандартный способ для личных приложений без App Store.

## Быстрый старт

### 1. Загрузить на GitHub

На Mac или Windows (после установки Git):

```bash
cd путь/к/Ежедневник
git init
git add .
git commit -m "Initial commit: iOS ежедневник"
git branch -M main
git remote add origin https://github.com/ВАШ_ЛОГИН/ezhnedevnik.git
git push -u origin main
```

Сначала создайте пустой репозиторий на [github.com/new](https://github.com/new) с именем `ezhnedevnik`.

### 2. Открыть в Xcode

```bash
git clone https://github.com/ВАШ_ЛОГИН/ezhnedevnik.git
cd ezhnedevnik
open Ezhnedevnik.xcodeproj
```

### 3. Настроить подпись (Signing)

1. В Xcode выберите проект **Ezhnedevnik** слева
2. Вкладка **Signing & Capabilities**
3. Включите **Automatically manage signing**
4. Выберите свой **Team** (Apple ID)
5. При необходимости смените **Bundle Identifier** на уникальный, например: `com.вашеимя.ezhnedevnik`

### 4. Установить на iPhone

1. Подключите iPhone к Mac
2. На iPhone: **Настройки → Конфиденциальность → Режим разработчика** (если iOS попросит)
3. В Xcode вверху выберите ваш iPhone как устройство
4. Нажмите **Run** (▶)
5. На iPhone: **Настройки → Основные → VPN и управление устройством** → доверьте своему Apple ID

Приложение будет работать ~7 дней, потом нужно снова нажать Run в Xcode (бесплатный Apple ID). С платным Apple Developer ($99/год) — до 1 года.

## Как пользоваться

1. Откройте приложение — увидите текущую неделю
2. Стрелки ← → — другая неделя, кнопка **Сегодня** — вернуться
3. Нажмите на час (например, `14`) — откроется редактор записи
4. Добавьте заметку и/или **Выбрать контакт** из телефонной книги
5. Иконка человека вверху справа — статус доступа к контактам

В альбомной ориентации неделя отображается ближе к макету на фото: 3 дня сверху, 4 снизу.
